# Java User Groups in Italy

Community is the heart of the Java ecosystem. Joining a **Java User Group (JUG)** is a great way to stay updated, share knowledge, and network with fellow professionals. Below is a list of active JVM-focused communities in Italy:

* **[JVM User Group "JUG Genova"](https://www.juggenova.it/)** - The community where I am co-founder.
* **[JVM User Group "JUG Milano"](https://www.jugmilano.it/)**
* **[Java User Group "JUG Torino"](https://jugtorino.org/)**

*Get involved and start sharing!*
