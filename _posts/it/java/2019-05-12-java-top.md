> “Il buon codice è la migliore documentazione di se stesso.”
> – Steve McConnell

Uno dei motivi del grande successo di Java come linguaggio di programmazione è che ha dominato lo spazio Enterprise fin dalla sua nascita e continua a farlo. 

In precedenza J2EE e oggi JEE 8 (al momento della stesura JDK versione 12) sono le piattaforme più popolari in aree come le banche e le assicurazioni.

Java domina ancora il mondo Android, nonostante soluzioni javascript come Ionic e il nuovo mondo IoT. Per quanto riguarda il mondo Android, si è aggiunto da tempo il linguaggio [Kotlin](https://kotlinlang.org/), ma è ancora ben lontano dal sorpasso. 

Uno dei maggiori vantaggi delle ultime versioni di Java EE è la riduzione del codice boilerplate. Gli sviluppatori possono scrivere meno codice, avere un supporto migliore per le applicazioni Web e i framework più recenti, oltre a una maggiore scalabilità e a un set più ricco di funzionalità che semplificano compiti che in precedenza erano più complessi (e noiosi). 

Una delle sfide più grandi per gli sviluppatori Java EE è tenersi aggiornati sui recenti sviluppi tecnici nella loro area di interesse.
